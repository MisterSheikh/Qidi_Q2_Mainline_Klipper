# Personal Config Reference

I've provided my own personal config as a refrence. Note that you WILL need to adopt this for your own purposes as I'm using the 2 pin connector on the toolhead board for my part cooling fan, etc.

You can use my load cell calibration values but I would highly encourage calibrating it yourself.
I do not know if they'll work for your machine or not.

```ini
[load_cell_probe]
counts_per_gram = 179.39500
reference_tare_counts = -1113211
```
