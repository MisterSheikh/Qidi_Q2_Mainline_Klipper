# Stock Qidi Q2 Klipper Firmware Dumps

Here are the full firmware dumps that I took from the mainboard and toolhead board MCUs
as a backup. You can use them to restore back to the stock firmwares by flashing with
st-flash from st-link tools, write to `0x8000000` for the respective MCU.

## Note: This should work but I haven't verified it personally for the TH board MCU.
